SQL提供了2个常用函数，用于条件判断。

| 函数                                                         | 说明                                                        | 示例                          |
| ------------------------------------------------------------ | ----------------------------------------------------------- | ----------------------------- |
| IF(条件1,值1,值2)                                            | 条件1成立，则返回值1<br />条件1不成立，则返回值2            | if(num>0, 1, 0) as `是否退货` |
| case when 条件1 then 值1<br />when 条件2 then 值2<br />else 值3<br />end | 条件1成立，返回值1<br />条件2成立，返回值2<br />否则返回值3 |                               |

- 判定条件可以是1个，也可以是多个；
- 多个判定条件表达式之间使用and & or连接；
